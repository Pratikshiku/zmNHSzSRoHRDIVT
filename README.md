## College Student Course Selection Leave Grade Course Comprehensive Management System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/5f8a0c7f57974470af9ca00197db6f2d/ghb20250919) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can be used to run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Feature Description
The system is divided into (roles): regular users (students), teachers, administrators.

**Regular User Features:**
1. Login
2. Registration
3. Homepage
4. Forum
5. Course Information
6. Announcement Information
7. College Display
8. Public Resources
9. Personal Center
10. Student Grades
11. Activity Registration
12. Student Leave Application
13. Course Selection Information
14. Credit Increase Records
15. Exchange Application


**Teacher Features:**
1. Login
2. Registration
3. Change Password
4. Personal Information
5. Course Information
6. Activity Registration Management
7. Student Grade Management
8. Student Leave Application Management
9. Course Selection Information Management
10. Exchange Application Management
11. Credit Increase Records
12. College Display Management
13. Public Resource Management
14. Forum Management



**Administrator Features:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. Teacher Management
4. Student Management
5. Course Information Management
6. Activity Registration Management
7. Student Grade Management
8. Student Leave Application Management
9. Course Selection Information Management
10. Coach Application Management
11. Credit Increase Management
12. College Display Management
13. Public Resource Management
